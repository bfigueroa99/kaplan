\section{Lesson Objectives}
In this chapter, you will learn:
\begin{itemize}
    \item How to use ``used to'' for past habits and states
    \item How to use ``would'' for past repeated actions
    \item The difference between ``used to'' and ``would''
    \item Stative verbs that cannot be used with ``would''
\end{itemize}

\section{Key Concepts: Past Habits}

We use two structures to talk about things we did in the past but don't do now.

\begin{vocabbox}[title=Used To vs. Would]
\begin{itemize}
    \item \textbf{Used to + Verb:} For past habits AND states.
    \item \textbf{Would + Verb:} ONLY for past repeated actions.
\end{itemize}
\end{vocabbox}

\section{Grammar Focus: The Rules}

\begin{grammarbox}[title=Rule 1: Action Verbs]
For repeated actions (play, go, visit, eat), you can use both:
\begin{itemize}
    \item I \textbf{used to} play football. \checkmark
    \item I \textbf{would} play football. \checkmark
\end{itemize}
\end{grammarbox}

\begin{grammarbox}[title=Rule 2: Stative Verbs]
For states (be, have, live, love, know), use ONLY \textbf{used to}:
\begin{itemize}
    \item I \textbf{used to} be shy. \checkmark
    \item I \textbf{would} be shy. \textbf{X} (Incorrect)
    \item I \textbf{used to} have a car. \checkmark
    \item I \textbf{would} have a car. \textbf{X} (Incorrect)
\end{itemize}
\end{grammarbox}

\section{Key Takeaways}
\begin{itemize}
    \item \textbf{Used to} works for everything (habits and states).
    \item \textbf{Would} only works for actions (running, playing), not states (being, having).
    \item Negative: \textbf{didn't use to} (no 'd').
    \item Question: \textbf{Did you use to...?}
\end{itemize}
