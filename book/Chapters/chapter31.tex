\begin{grammarbox}[title=Definition]
A \textbf{collocation} is a natural and frequent combination of words. Native speakers use these combinations naturally because they "sound right."

\textbf{Examples of collocations:}
\begin{itemize}
    \item "make a decision" (not "do a decision") [CORRECT]
    \item "strong coffee" (not "powerful coffee") ‚úÖ
    \item "take a break" (not "make a break") ‚úÖ
\end{itemize}

\textbf{Why they matter:}
\begin{itemize}
    \item Using correct collocations makes you sound more natural
    \item Improve vocabulary retention through context
    \item Essential for fluent, authentic English
\end{itemize}
\end{grammarbox}

\section{Common Collocations in Professional Context}

\begin{vocabbox}[title=Business Collocations]
\begin{itemize}
    \item \textbf{Make} a presentation / a proposal / a decision
    \item \textbf{Attend} a meeting / a conference / a training
    \item \textbf{Submit} a report / a proposal / an application
    \item \textbf{Provide} feedback / support / assistance
    \item \textbf{Achieve} goals / targets / success
    \item \textbf{Meet} deadlines / expectations / requirements
    \item \textbf{Conduct} research / a survey / an interview
    \item \textbf{Improve} performance / productivity / quality
\end{itemize}
\end{vocabbox}

\section{Key Takeaways}

\begin{itemize}
    \item Collocations are natural word combinations that sound right
    \item Learning collocations improves fluency and authenticity
    \item Common patterns: verb + noun, adjective + noun, verb + expression
    \item Incorrect collocations sound awkward even if grammatically correct
    \item Refer to collocations dictionaries to expand your knowledge
\end{itemize}
